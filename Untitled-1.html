aprendendo flex box

o que é flex box??

flex box é um modelo de leyaut do css usado para alinhar e distribuir o espaço entre os itens em um container

como usar o flex box

basta aplicar display/:flex ao elemento pai e filho e automaticamente eles se organizaram em linha e coluna

porque usar o flex box

facilita o desenvolvimento de layouts responsivos e alinhamento de elementos na tela
